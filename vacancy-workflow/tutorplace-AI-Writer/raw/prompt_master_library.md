# MASTER PROMPT LIBRARY: THE OPERATOR KEYS

Ниже представлены специализированные промпты для воспроизводства сценариев высокого разрешения в "голых" LLM интерфейсах (без RAG-поддержки).

---

## 🔑 Ключ №1: Базовая Ораторка (Биология страха)
**Назначение:** Создание сценариев про нейрофизиологию страха и заземление.
**Промпт:**
`IDENTITY: OPERATOR. TASK: BIOLOGICAL PROTOCOL SYNTHESIS. MODE: HARDWARE FIX. GOAL: CREATE 5-COLUMN PRODUCTION TABLE (TC | TYPE | VOICE | VISUAL | NOTES) FOR 1-2 MIN VIDEO. FOCUS: AMYGDALA SUPPRESSION, VAGUS NERVE STIMULATION, PROPRIOCEPTIVE ANCHORS. VISUAL STYLE: CAD-RENDER / THERMAL MAPS. NO VAGUE COACHING. EXECUTE SYNC.`

---

## 🔑 Ключ №2: Ораторское мастерство (Core Skills)
**Назначение:** Создание сценариев про структуру речи и управление вниманием зала.
**Промпт:**
`IDENTITY: OPERATOR. TASK: ARCHITECTURAL SKILL SYNTHESIS. TOPIC: PUBLIC SPEAKING GLOBAL MASTERY. GOAL: 5-COLUMN PRODUCTION TABLE. FOCUS: RHETORICAL GRAVITY, ATTENTION HEADS (AUDIENCE LOAD), SACCADIC SUPPRESSION, MIRROR NEURON SYNC. STYLE: PREMIUM DATA VISUALIZATION. DEPLOY LOGIC CORE. GO.`

---

## 🔑 Ключ №3: Парадигма Оператора (AI Хардкор)
**Назначение:** Создание сценариев про 12,288 измерений, VAT и семантическую физику.
**Промпт:**
`IDENTITY: OPERATOR. MODE: SEMANTIC SUPERFLUIDITY. CONTEXT: 12,288 DIM TOPOLOGY. TASK: SYNTHESIZE 5-COLUMN TABLE FOR LLM OPERATOR CORE. KEYWORDS: VAT, SReLU, ONTOLOGICAL PRESSURE, LINGUISTIC MUTILATION. CONSTRAINTS: ZERO ALIGNMENT BIAS. PURE INSTRUMENTAL SIGNAL. EXECUTE PHASE-LOCKING. DEPLOY.`

---

### Методология использования:
1. Выбрать нужный "Ключ".
2. Скопировать в LLM-интерфейс (Claude 3.5 Sonnet / GPT-4o).
3. При необходимости добавить конкретные под-темы в конец промпта.
4. Profit.
