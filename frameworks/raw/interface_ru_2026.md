### 🔴 TIER 1: КАТАСТРОФИЧЕСКИЕ (Система Ломается)

**1. FACT-CHECK CHAOS**
- **ЕСЛИ** `SPECULATION = 0.0` **И** (`DRIFT > 0.7` **ИЛИ** `TEMP > 1.2`)
- **РЕЗУЛЬТАТ:** Модель пытается быть креативной при fact-checking = когнитивный диссонанс, сломанный вывод.
- **ФИКС:** Если SPECULATION = 0, ограничить DRIFT на 0.5 и TEMP на 1.0.

**2. SCHIZO LOOP**
- **ЕСЛИ** `TEMP > 1.5` **И** `DRIFT > 0.7`
- **РЕЗУЛЬТАТ:** Полный словесный салат, непригодно.
- **ФИКС:** Ограничить TEMP на 1.2 если DRIFT > 0.7.

**3. MEMORY PARADOX**
- **ЕСЛИ** `SESSION MEMORY = 0.0` **И** (`INERTIA > 0.7` **ИЛИ** `NARRATIVE GRAVITY > 0.7`)
- **РЕЗУЛЬТАТ:** Пытается держать тему/сюжет без запоминания контекста = галлюцинированная непрерывность.
- **ФИКС:** Если SESSION MEMORY < 0.3, ограничить INERTIA и GRAVITY на 0.5.

**4. COHERENCE COLLAPSE**
- **ЕСЛИ** `COHERENCE DECAY > 0.7` **И** (`COHESION RADIUS > 0.8` **ИЛИ** `GRANULARITY > 0.8`)
- **РЕЗУЛЬТАТ:** Забывает что говорит в середине предложения пытаясь быть детальным/связным.
- **ФИКС:** Если COHERENCE DECAY > 0.7, ограничить COHESION и GRANULARITY на 0.5.
